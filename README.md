# Vintage Dial

Vintage Dial is a feature-rich e-commerce website developed using PHP and MySQL for a premium watch brand. The project is designed to provide customers with a secure, responsive, and user-friendly online shopping experience while offering administrators a centralized system for managing all aspects of the online store.

## Features

- Public storefront with product browsing and category filtering
- Product detail pages with images, specifications, pricing, and availability
- Shopping cart with quantity updates and checkout flow
- User registration, login, profile management, and order history
- Admin panel to manage products, categories, orders, customers, reviews, and contact messages
- Image upload support for product and brand assets
- Dynamic homepage content stored in the database
- Responsive frontend built with HTML5, CSS3, JavaScript, Bootstrap, and custom styles

## Technology Stack

- PHP
- MySQL
- HTML5
- CSS3
- JavaScript
- Bootstrap

## Description

The admin dashboard simplifies store management with CRUD operations for products, categories, and customer orders. The site content can be updated from the database without editing source code, making it easy to keep banners, featured sections, and other shared content current.

This application demonstrates practical implementation of authentication, database management, session handling, file uploading, and responsive design in a full e-commerce website.

## Setup

1. Import the `vintage_dial` database schema into MySQL.
2. Update database credentials in `admin/config/db.php` if needed.
3. Place the project in the web server root (e.g. `htdocs` for XAMPP).
4. Open the site in the browser and use the admin panel to manage store content.

## Notes

- The project is suitable for real-world e-commerce operations and can be extended with payment integration, search enhancements, and improved security features.
